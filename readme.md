# Ikizler Bale Akademi Theme

Bu klasor, `ozelikizlerbalekursu.com` icin hazirlanan block theme tabanli WordPress tema iskeletidir.

## Faz 1 kapsaminda gelenler

- `theme.json` tabanli tasarim sistemi
- Full Site Editing uyumlu temel template yapisi
- Header / footer template part'lari
- Ana sayfa icin premium baslangic pattern'leri
- Editor tarafinda daha tutarli gorunmesi icin editor stili

## Kurulum

1. Bu klasoru WordPress kurulumundaki `wp-content/themes/ikizler-bale-akademi` altina tasiyin.
2. WordPress yonetim panelinden temayi etkinlestirin.
3. `Gorunum > Editor` altindan navigasyon, sayfa icerikleri ve template'leri duzenleyin.
4. `Ayarlar > Okuma` altindan ana sayfayi statik sayfa olarak tanimlayin.
